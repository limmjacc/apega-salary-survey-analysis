## Active Data Files (8.05 KB)

### salary_master.json (3.38 KB)
**Purpose:** Consolidated dataset containing all years  
**Content:** 
- 5 years of salary data (2020, 2021, 2022, 2023, 2024)
- Career levels: P1-P6, M1-M5 (per profession)
- Professions: Engineers (ENG) & Geoscientists (GEO)
- Organization stats, demographic data

**Used by:** `forecast_salaries.py`, `generate_reference_tables.py`

### salary_data_2021.json (1.15 KB)
**Purpose:** Year-specific 2021 salary data  
**Content:**
- 11 ENG levels (P1-P6, M1-M5)
- 11 GEO levels (P1-P6, M1-M5)
- Organization count: 169
- Incumbents: 10,821
- Data source: Manual recovery from font-corrupted PDF

**Generated by:** `extract_2021_manual.py`

### salary_data_2024.json (1.35 KB)
**Purpose:** Year-specific 2024 salary data  
**Content:**
- 11 ENG levels (P1-P6, M1-M5)
- 11 GEO levels (P1-P6, M1-M5)
- Organization count: 184
- Incumbents: 11,945
- Work arrangements: 28% remote/hybrid, 72% in-office
- Data source: Manual recovery from image-based PDF

**Generated by:** `extract_2024_manual.py`

### salary_forecasts_2024_2030.json (2.17 KB)
**Purpose:** Salary projections through 2030  
**Content:**
- 7-year forecasts (2024-2030)
- All career levels (P1-P6, M1-M5)
- Both professions (ENG, GEO)
- Regression-based predictions

**Generated by:** `forecast_salaries.py`

## Usage Examples

### Run Forecasting
```bash
python scripts/forecast_salaries.py
```
Generates updated forecasts and visualizations using all available data.

### Verify Data Integrity
```bash
python verify_data.py
```
Validates all JSON files and reports data consistency.

### Extract Year-Specific Data
```bash
# 2021
python scripts/extract_2021_manual.py

# 2024
python scripts/extract_2024_manual.py
```
Regenerates year-specific files (updates salary_master.json).
